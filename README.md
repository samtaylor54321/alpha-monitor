# Alpha Monitor

Alpha Monitor is a personal project for testing investment ideas and tracking them through a simple dashboard.

The aim is to monitor signals, strategies, and performance over time, and see whether ideas hold up in practice rather than just in backtests.

alpha-monitor/
├── data/ # Data sources and processed datasets
├── strategies/ # Strategy logic
├── signals/ # Indicators and signals
├── backtests/ # Backtesting code
├── dashboard/ # Dashboard application
├── notebooks/ # Exploratory analysis
└── README.md
